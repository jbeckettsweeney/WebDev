Wavefunction Collapse is an algorithm which takes in a bitmap and, after having dissected that bitmap, procedurally rearanges pieces in order to create a new bitmap which follows similar rules. This has numerous applications such as procedurally generating levels in a video game and generating art. With one's own implementation, the manner in which generation occurs can be modified, but with this tool you can experience this implementation by uploading an image, generating, and then downloading your new image. Enjoy!
